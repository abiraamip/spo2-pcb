# spo2-pcb
PCB design of DIY SpO2 module

An SpO₂ module is a sensor module used to measure:
SpO₂ (Peripheral Oxygen Saturation) – the percentage of oxygen-carrying hemoglobin in the blood.
Heart Rate (Pulse Rate) – the number of heartbeats per minute (BPM).

How it Works
The module shines two different wavelengths of light through a finger or earlobe:
Red light (~660 nm)
Infrared light (~940 nm)
Oxygenated and deoxygenated hemoglobin absorb these wavelengths differently. By analyzing the amount of light absorbed, the sensor estimates:
Blood oxygen saturation (SpO₂)
Pulse rate
